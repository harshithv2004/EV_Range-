How It Works

01

Data Collection

Trained on 194 real-world EVs from the EV Database, covering 14 technical specifications per vehicle.

02

Feature Engineering

Correlation analysis selected the most impactful features. Drive type is label-encoded; all features are L2-normalized.

03

Model Selection

Linear Regression outperformed Decision Trees and Random Forests with the lowest MAE (~8.95 km) on the test set.

04

Multi-Output Prediction

The model predicts 7 range values simultaneously: electric range + city/highway/combined for cold and mild weather.
